# Conventions

Read the relevant convention file before adding or modifying documents in the `.agents/` directory.

## Convention Files

| File | When to read |
|------|-------------|
| [EPIC.md](EPIC.md) | Adding or modifying an epic (`EPIC_*.md` in the root directory) |
| [SPEC.md](SPEC.md) | Adding or modifying a spec (`spec/*.md`) |
| [ADR.md](ADR.md) | Adding or modifying an architecture decision record (`adr/*.md`) |
| [REF.md](REF.md) | Adding or modifying a reference document (`ref/*.md`) |
