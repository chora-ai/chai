# Third-Party OpenAI GPT

## Scope

Third-party OpenAI API runs for GPT models.

## Models

- `gpt-5.2`
- `gpt-5.1`
- `gpt-5.1-mini`
- `gpt-5-mini`

## Setup

- `agents.defaultProvider`: `openai`
- `agents.defaultModel`: one model from the list above
- `OPENAI_API_KEY` or `providers.openai.apiKey`

## Procedure

Follow the shared protocol in [README.md](README.md): message sequence, expectations, and run procedure.
