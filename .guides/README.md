# User Guides

This directory holds **task-oriented guides** for using Chai: how things work on disk, which commands to run, and what to avoid. The tone is closer to user documentation than to design specs in `.agents/` or manual test scripts in `.journey/`.

Guides are written against the **current** behavior of the CLI and libraries; if something feels awkward, that is a signal to improve the product—guides may be updated when behavior changes.
